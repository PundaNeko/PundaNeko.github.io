main_async.py — parallel scraping core
async def scrape_title(session, link):
    async with session.get("https://runescape.wiki" + link) as response:
        html = await response.text()
        soup = BeautifulSoup(html, "html.parser")
        title = soup.find("span", class_="mw-page-title-main")
        return link, title.get_text(strip=True) if title else "No title"

async def main(links):
    async with aiohttp.ClientSession() as session:
        tasks = [scrape_title(session, link) for link in links]
        results = await asyncio.gather(*tasks, return_exceptions=True)

Project 05 — HTML / CSS / JS
Interactive Portfolio Site
Book-style UI with flashlight effect and puzzle mechanic
JavaScript CSS Animations HTML

A personal website built as an interactive book. Pages turn via a CSS scroll-timeline carousel with a sprite-animated book background.
