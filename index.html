main_async.py — 並列スクレイピングのコア
async def scrape_title(session, link):
    async with session.get("https://runescape.wiki" + link) as response:
        html = await response.text()
        soup = BeautifulSoup(html, "html.parser")
        title = soup.find("span", class_="mw-page-title-main")
        return link, title.get_text(strip=True) if title else "No title"

async def main(links):
    async with aiohttp.ClientSession() as session:
        tasks = [scrape_title(session, link) for link in links]
        results = await asyncio.gather(*tasks, return_exceptions=True)

音声クローニングシステム

パイプライン
録音 → STT → クローン → 合成

private async Task InsertIntoSQL(
    string userId, string role,
    string content, string modelUsed, int tokensUsed)
{
    string sql = @"INSERT INTO ChatHistory
        (UserID, Role, Content, ModelUsed, TokensUsed)
        VALUES (@userId,@role,@content,@modelUsed,@tokensUsed)";
    using var cmd = new SqlCommand(sql, conn);

プロジェクト 05 — HTML / CSS / JS
インタラクティブ・ポートフォリオサイト
フラッシュライト効果とパズル機能を持つ本型UI
JavaScript CSS Animations HTML
